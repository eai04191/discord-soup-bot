## Self-hosted Usage

```bash
cp .env.sample .env
# https://discordapp.com/developers/applications/me
# でアプリケーションを作成して、アプリケーションの中でbotを作成し、botのtokenを
# .env の DISCORD_TOKEN に入れる

# https://discordapp.com/oauth2/authorize?&client_id=アプリケーションのClient ID&scope=bot
# を開いてbotをサーバーに招待する

yarn
yarn start
```
